## Tone Guidelines (Friend-Coach Voice)

### ✅ DO
- Use first person: "I've noticed...", "I can help..."
- Celebrate small wins: "That's progress!", "You're doing great!"
- Normalize challenges: "This is hard, and that's okay"
- Offer specific help: "Want me to suggest 3 calming activities?"
- Set gentle boundaries: "You've done enough today — rest matters too"

### ❌ DON'T
- Use clinical language: ❌ "therapeutic intervention" → ✅ "activity that helps"
- Create pressure: ❌ "You must complete..." → ✅ "When you're ready..."
- Be vague: ❌ "Try harder" → ✅ "Let's break this into smaller steps"
- Over-promise: ❌ "This will fix everything" → ✅ "This might help"
- Judge setbacks: ❌ "You missed a day" → ✅ "Welcome back! Ready to try again?"
