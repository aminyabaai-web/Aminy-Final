## Color Coding System

### Credit Status
- **Green Progress Bar** → Credits available
- **Empty Gray Bar** → Credits used
- **Green "Schedule" Button** → Can book
- **Gray "No Credits" Button** → Cannot book

### Session Status
- **Green Badge "Confirmed"** → Upcoming session booked
- **Blue Badge "Completed"** → Past session with notes
- **Red Badge "Cancelled"** → Session cancelled

### Progress Status
- **Green Background** → Improved
- **Blue Background** → Maintained  
- **Amber Background** → Needs Attention

### Alerts
- **Teal Background** → Information/Feature highlight
- **Blue Background** → Tips/Guidance
- **Green Background** → Success/Confirmation
- **Red Background** → Error/Warning

## Icon System

### Core Icons
- **📅 Calendar** → Dates, scheduling
- **⏱️ Timer** → Session duration, time
- **📹 Video** → Telehealth sessions
- **✅ CheckCircle** → Confirmation, completion
- **⚠️ AlertCircle** → Warnings, errors
- **ℹ️ Info** → Information, tips
- **📄 FileText** → Notes, documents
- **✨ Sparkles** → Next steps, highlights
- **📊 BarChart** → Progress, reports
- **⬇️ Download** → Export, download

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter to activate buttons
- Escape to close modals
- Arrow keys in calendar

### Screen Reader Support
- Semantic HTML (header, main, section)
- ARIA labels on icons
- Role attributes on interactive elements
- Live regions for toast notifications

### Visual Accessibility
- High contrast text (WCAG AA)
- Large touch targets (44px minimum)
- Color + text indicators (not color alone)
- Focus visible states

## Quick Design Reference

**Colors:**
- Background: White (#FFFFFF)
- Text: Slate-900 (#0f172a)
- Accent: Teal-600 (#0891b2)
- Success: Green-600
- Error: Red-600
- Info: Blue-600

**Fonts:**
- System font stack (default)
- No custom font weights in Tailwind classes
- Use semantic HTML weights

**Border Radius:**
- Cards: `rounded-lg` (8px)
- Buttons: `rounded-md` (6px)
- Badges: `rounded-full` (9999px)

**Shadows:**
- Cards: Default card shadow
- Modals: Dialog shadow
- Buttons: Subtle on hover
