# Badge — soft identity pill

Carries meaning, never decoration. Tone maps to the earned-state palette.

```jsx
<Badge tone="teal">Morning routine</Badge>
<Badge tone="win" icon="⭐">Win</Badge>
<Badge tone="grow">3-day streak</Badge>
<Badge tone="care">Provider review</Badge>
```

Tones: `teal` (default) · `navy` · `win` (amber) · `grow` (green) · `care` (violet/provider).
