# Nudge — the signature proactive card

A warm rounded (Fredoka) tip from Aminy with a teal left-stripe. Validate first, then offer.

```jsx
<Nudge actions={<><Button size="sm">Try it today</Button><Button size="sm" variant="ghost">Not this one</Button></>}>
  Mornings are rough — you already know. Try teeth before dressed this week. Small change, big difference.
</Nudge>
```

The tip is always the rounded brand font (Fredoka). Never stack stock icons inside the body.
